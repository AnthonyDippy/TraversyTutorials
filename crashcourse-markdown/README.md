## Notes

### Markdown

* Lightweight markup language
* Plain text syntax
* Can be converted to html/xhtml
* Readability and ease of use
* Different flavors/additional features

### Use

* README.md files (Github)
* Personal notes
* Forum/blog posts
* Static website generators

### Features

* Headings
* Lists
* Code Blocks
* Blockquotes
* Images
* Links
* Etc
